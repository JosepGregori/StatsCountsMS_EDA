1 - INSTALLATION

    StatsCountsMS_EDA is an R script which may be run directly on R provided
  that you have installed R v 3.3.0 or higher, the package msmsEDA and
  its dependencies, and the packages gWidgets and RGtk2.

2 - USER MANUAL

    Please see the user manual provided. 
    Further help may be found in the vignette and manual of the msmsEDA
    package.
